🎮 Turtle Game - Escape da Areia
Um jogo simples desenvolvido em Python utilizando a biblioteca Turtle.

📖 Sobre o jogo.
O objetivo é controlar a tartaruga e guiá-la da areia até o mar, desviando de obstáculos e alcançando a área segura.
O jogador utiliza as teclas de direção do teclado para movimentar a tartaruga pelo cenário.

🕹️ Controles
Tecla	Ação
⬆️ Seta para cima	Mover para cima
⬇️ Seta para baixo	Mover para baixo
⬅️ Seta para esquerda	Mover para esquerda
➡️ Seta para direita	Mover para direita

🚀 Tecnologias utilizadas
Python
Turtle Graphics

▶️ Como executar
Clone o repositório:
git clone https://github.com/Hipolito33/turtleGame-python.git
Acesse a pasta do projeto:
cd turtleGame-python
Execute o arquivo principal:
python main.py

🎯 Objetivo
Levar a tartaruga da areia até o mar utilizando as setas do teclado
